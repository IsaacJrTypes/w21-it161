Forms Research

Forms play a big role in how websites receive input from users. In the early years of the internet, forms were limited to only submitting search requests (Illustrated History). Without forms, we could only read websites and have no simple way to buy goods and services online. Many websites wouldn’t be able to provide online profiles if it weren’t for forms. In the past, websites used to have the user download a form, print, fill out and mail it with a postage stamp (Illustrated History). Online forms have changed how users interact with websites.

When users submit information in forms, servers receive the data all at once. Input elements allow users to designate a value or provide a space to write in data. Using HTTP, the collected information is delivered through two methods: GET or POST (woods). GET will send data input to a server visible to the user in the URL. POST hides users’ input in the URL but may require additional scripting language (Using the Post Method). For simplified break down which method to use, watch the following video:

Since forms are like any other block-level element, CSS provides aesthetic control in presenting input elements to users. With CSS, you can add an image in the background of a text input box (Bitdegree).

UxMatters.com found that by placing labels of a form input above the input field, reduced the extent the user had to move their eyes (Penzo). (Image Below) The larger the eye movement meant the user applied more effort to digest what was being presented. Good design communicates information to the user efficiently without straining the user.

Websites would not be as interactive as they are now if forms didn’t exist. Third‑party applications would slow down the user experience to provide the same level of service form provides. Forms provide an efficient and effective way for websites to collect input from its users that benefits both parties.

How has social media platforms become a market force that is too big to ignore? While reading Social intelligence by Daniel Goleman, I learned about the power of mirror neurons. The book goes into how humans use mirror neurons throughout the brain, to learn survival skills by observing others. Our mirror neurons activate whenever we see a human emote on social media. Social media platforms allow humans to share their experiences and knowledge with ease. If you want your (or friend’s) website to be part of that experience, social media optimization is essential.
